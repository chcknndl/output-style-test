카운터 앱 코드 핵심 설명
=========================

[ 개념 설명 ]

- let count = 0
  숫자를 기억하는 변수 (저장 공간)
  프로그램이 실행되는 동안 현재 카운트 값을 담아둡니다.

- id="count"
  JavaScript가 HTML 요소를 찾을 때 사용하는 이름표
  HTML 태그에 id를 붙이면 JS에서 그 요소를 쉽게 찾을 수 있습니다.

- document.getElementById(...)
  id로 HTML 요소를 찾는 함수
  예: document.getElementById('count') → id가 "count"인 태그를 가져옵니다.

- .textContent = count
  찾은 요소의 텍스트를 변경
  화면에 보이는 숫자를 count 변수의 값으로 바꿉니다.

- onclick="increase()"
  버튼 클릭 시 실행할 함수를 지정
  버튼을 누르면 괄호 안에 적힌 함수가 자동으로 실행됩니다.


[ 동작 흐름 ]

버튼 클릭 → 함수 실행 → count 변수 변경 → updateDisplay()로 화면 갱신
